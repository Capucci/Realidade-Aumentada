O que é o MindAR?

MindAR é uma biblioteca JavaScript de código aberto que permite criar experiências de realidade aumentada diretamente no navegador, sem necessidade de aplicativos.

Como usar:

  1. Clique em "Iniciar Câmera" para ativar a câmera
  2. Aponte para a imagem alvo (ao lado)
  3. Veja o modelo 3D aparecer em realidade aumentada!

Vantagens do MindAR:

  • Funciona diretamente no navegador
  • Não requer instalação de aplicativos
  • Compatível com dispositivos móveis e desktop
  • Fácil implementação com HTML
  • Suporte a múltiplos alvos
  • Rastreamento em tempo real
